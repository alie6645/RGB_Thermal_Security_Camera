Contains scripts used for RGB to thermal neural network translation \[paper](https://www.overleaf.com/read/xwymdkgvqxsc#39ed2b)



bright-only: contains code used for dark-excluding dataset



NAFNET\_model: contains code used in brightened RGB dataset



camera\_scripts: contains code found on Orange Pi in the RGB-thermal camera



nafnet: defines nafnet architecture



pix2pix: defines pix2pix architecture



unet: defines unet architecture



samples: example results



video: code for video demos
